360yunpan
=========

360YunPan Command-line tools, support: Linux Mac Windows


##功能：
* 云盘登陆
* 云盘文件下载
* 云盘离线下载
* 命令行工具
* 云盘文件上传（开发中）

```
  用法: ./cli.py -o http://xxx.com/xxx.tar.gz

  例子:
        开始离线下载
        ./cli.py -o http://xxx.com/xxx.tar.gz
        ./cli.py --offline="http://xxx.com/xxx.tar.gz"

  -h, --help 查看帮助
  -o, --offline 开始离线下载
  -t, --task 查看离线下载队列
```


##目标：
* 实现跨平台的360云盘命令行工具
